# David Friedrich | UI Engineer

Personal portfolio and resume site. Built to be lightweight, simple to maintain, and fast.

## Tech Stack
- **UI:** React 18 / TypeScript
- **Environment:** Node.js (SSR)
- **Styles:** SCSS / FontAwesome Pro

## Project Structure
- **No CMS:** Content is managed directly in the code for simplicity and performance.
- **SSR:** Server-side rendered for better initial paint and accessibility.

## Local Development

### Setup
npm install

### Run
npm run start dev
